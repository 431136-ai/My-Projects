## Chess

This project is a chess game that includes both a two-player mode and a single-player mode against an AI. The goal of the project was to create a fully functional chess game where players can either compete with another player or play against the computer.

My role in the project was coding the chess pieces and implementing the rules of the game. I also developed the logic that controls how each piece moves and interacts on the board, ensuring that the gameplay follows standard chess rules.
